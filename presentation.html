Why FOSS

GPLv3 means anyone can:

  • run the software
  • study the software
  • share the software
  • modify the software

This encourages:

  • sharing of data, formats, software, etc.
  • continuity between possibly distant projects
  • fewer conflicting standards (e.g. file formats)
  • collaboration between people doing different things
  • optimal usage of limited resources
